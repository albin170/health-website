## Getting Started

First, install dependencies:

```bash
npm install
```

Then run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Host the app locally (LAN accessible)

To host the app so other devices on your network can access it:

```bash
npm run host
```

This runs Next.js on `0.0.0.0:3000`.

## Production hosting

Build and start the production server:

```bash
npm run build
npm run start
```
